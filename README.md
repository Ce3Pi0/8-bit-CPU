# 8-bit Computer

8-bit computer inspired by <a href="https://www.youtube.com/watch?v=HyznrdDSSGM&list=PLowKtXNTBypGqImE405J2565dvjafglHU&index=2">Ben Eater's YouTube video series</a> simulated using <a href="https://github.com/logisim-evolution">logisim evolution</a>

Allowed built in components:

- Logic Gates: (AND, OR, NOT, XOR, NAND, NOR, XNOR)
- Other: (Ground, Power, Constant, Input, Output, Splitter, Tri State Buffer)
- Flip Flops: (D, JK, T, SR)
